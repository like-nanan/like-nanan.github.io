I. Model Deployment and Usage

Online Model API: OpenAI

Account settings

Register an account and obtain the API KEY for invoking the large model through the API.

Configure the API Key to the environment variable

Run the following command in CMD.

setx DASHSCOPE_API_KEY "YOUR_DASHSCOPE_API_KEY"

Install openai

Call the model on the Alibaba Cloud Bailian platform through OpenAI’s Python SDK.

pip install openai

Call the Large Model API

Create a new file and name it.hello_qwen.py and Copy the following code into and save it.hello_qwen.py.

import os
from openai import OpenAI

try:
    client = OpenAI(
        # 若没有配置环境变量，请用阿里云百炼API Key将下行替换为：api_key="sk-xxx",
        api_key=os.getenv("DASHSCOPE_API_KEY"),
        base_url="https://dashscope.aliyuncs.com/compatible-mode/v1",
    )

    completion = client.chat.completions.create(
        model="qwen-plus",  # 模型列表：https://help.aliyun.com/zh/model-studio/getting-started/models
        messages=[
            {'role': 'system', 'content': 'You are a helpful assistant.'},
            {'role': 'user', 'content': '你是谁？'}
            ]
    )
    print(completion.choices[0].message.content)
except Exception as e:
    print(f"错误信息：{e}")
    print("请参考文档：https://help.aliyun.com/zh/model-studio/developer-reference/error-code")

Run via the command line or.python hello_qwen.py
