Developing an Application with the Aid of LLM

I. Background and Functionality

Design

This project is a GUI-based Airline Ticket Management System developed in Python. Flight booking is a typical application scenario in intelligent transportation. By implementing this project, we can understand the basic design of booking systems, and how to integrate GUI and file I/O in a Python application. It integrates academic interests in intelligent transportation systems and user interaction design.

Functionalities :

  • Query flights by ID, departure or destination

  • Add new flights

  • Delete existing flights

  • Purchase ticket from result list

  • Save/load flight data from local file

Challenges:

  • Managing GUI event logic

  • Handling real-time updates after purchase

  • Packaging Python GUI app on macOS ARM (M1/M2) with Python 3.13

2. Development Environment

Hardware / OS:

  • Apple M4 MacBook Air

  • macOS Sonoma 14.5

Tools and Compilers:

  • Python 3.13.4

  • tkinter (GUI)

  • PyInstaller 5.13.2 (for packaging)

AI Models:

  • OpenAI ChatGPT-4 (for code generation, debug, and GUI logic)

3. Development Process

Development Process

  1. Download the source file from GitHub
  2. Discussed core logic with LLM (flight class, system class)
  3. Used ChatGPT change C++ to python
  4. Used ChatGPT to auto-generate tkinter-based GUI skeleton
  5. Added dynamic update of ticket availability after purchase
  6. Debugged compatibility issues with PyInstaller on macOS M4 + Python 3.13
  7. Generated 100 test entries with AI and wrote to flight_data.txt
  8. Successfully compiled into main executable for macOS
